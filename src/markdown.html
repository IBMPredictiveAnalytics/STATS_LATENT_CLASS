STATS LATENT CLASS Extension Command

STATS LATENT CLASS MANIFESTVARS=variables* COVARIATES=variables CLASSES=n*

/OPTIONS REPETITIONS=m MAXITER=value TOL=value MISSING=INCLUDE** or LISTWISE

/OUTPUT GRAPHS=YES** or NO

/SAVE PREDCELLDS=dataset SCOREDS=dataset IDVAR=varname

/HELP

* Required
** Default

STATS LATENT CLASS /HELP prints this information and does nothing else.

STATS LATENT CLASS MANIFESTVARS = a b c CLASSES=3
/OPTIONS REPETITIONS = 5.

MANIFESTVARS and COVARIATES list the categorical and covariate variables. The categorical variables must have integer values starting at 1. If covariates are listed, latent class regression models are estimated. MANIFESTVARS is required, but COVARIATES is optional.

CLASSES is the number of latent classes

OPTIONS

In order to minimize the chance of finding only a local maximum of the likelihood, the procedure can reestimate at different random starting values. REPETITIONS, default 10, specifies how many sets to try.

MAXITER, default 1000, specifies the maximum number of iterations

TOL specifies the convergence criterion.

MISSING=INCLUDE uses all available nonmissing data. LISTWISE does listwise deletion of cases. Missing covariate values always cause the case to be deleted.

OUTPUT

GRAPHS controls whether graphs are displayed.

SAVE

PREDCELLS can specify a new dataset to be created holding the classification probabilities - observed and expected

SCOREDS can specify a new dataset to be created holding the scores for the model. If SCORES is specified, IDVAR is required and will be included in the dataset in order to facilitate merging with the input data. The scores dataset has variables, the ID, MaxProb, and MaxClass followed by probability variables for each class.
